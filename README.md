# Odin 2 Game Compatibility

Web version of the community-maintained [Odin 2 Compatibility Spreadsheet](https://docs.google.com/spreadsheets/d/1XaPYEyTinKk7F2uTfgSDpgEelhUYPFBWo7knX_3aQgw).

## Features

- Search and filter games by compatibility rating
- Browse by platform (GameCube, Wii, Switch, PS2, PS Vita, 3DS, Wii U, Android)
- Data updates automatically from the source spreadsheet

## Data Source

All data comes from the original spreadsheet maintained by the Odin community:

**[Odin 2 Compatibility Sheet](https://docs.google.com/spreadsheets/d/1XaPYEyTinKk7F2uTfgSDpgEelhUYPFBWo7knX_3aQgw)**

To report game compatibility, please submit directly to the spreadsheet.
